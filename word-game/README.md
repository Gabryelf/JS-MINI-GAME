## 📖 О проекте

**Words Constructor** - увлекательная браузерная игра в слова, где игрок собирает слова из букв, расположенных по кругу. Игра развивает словарный запас, логическое мышление и скорость реакции.

### ✨ Основные возможности

- 🎯 **Два режима игры**: Бесконечная игра и "Слово дня"
- 📈 **Прогрессивная сложность**: Уровни становятся сложнее по мере прохождения
- 🎨 **Красивый интерфейс**: Современный дизайн с анимациями
- 🔊 **Иммерсивный звук**: Генератор звуковых эффектов
- 💾 **Сохранение прогресса**: Продолжайте с того места, где остановились
- 📱 **Адаптивный дизайн**: Игра работает на всех устройствах

## 🎮 Как играть

### Основные правила:
1. **Цель**: Найти все слова из предложенного списка
2. **Управление**:
   - Кликайте по буквам на круге для составления слова
   - Нажмите центральную кнопку ✓ для проверки слова
   - Используйте Backspace для удаления последней буквы
   - Или нажмите кнопку ↺ для сброса ввода

3. **Баллы**:
   - +10 очков за каждую букву в слове
   - +100 бонусных очков за завершение уровня
   - Попытки восстанавливаются с каждым уровнем

### Режимы игры:

#### ♾️ Бесконечная игра
- Уровни с прогрессирующей сложностью
- Максимум 50 уровней
- Чем дальше - тем сложнее слова

#### 📅 Слово дня
- Новая тема каждый день
- Только одна попытка в сутки
- 4 различные темы:
  - 🐾 Животные
  - 🏙️ Города
  - 👨‍🍳 Профессии
  - 🍎 Еда

## 🛠️ Технические особенности

### Архитектура
- **Чистый JavaScript** без зависимостей
- **Модульная структура** для легкого сопровождения
- **Объектно-ориентированный подход**
- **Адаптивный дизайн** с CSS Grid и Flexbox

### Ключевые технологии:
- **Canvas API** для кругового ввода
- **Web Audio API** для генерации звуков
- **LocalStorage** для сохранения прогресса
- **CSS Animations** для плавных переходов

### Оптимизация:
- Минимальные HTTP-запросы
- Эффективное кэширование
- Адаптивные изображения
- Ленивая загрузка компонентов
